
Get Started
=====

## [功能简介]
- [x] 设计模式
- [x] 标签保护:文本(单行/多行)、单选、复选、下拉列表、文档块、日期
- [x] 数据异步加载
- [x] 超大数据加载
- [x] 编辑模式
- [x] 只读模式
- [x] 痕迹模式
- [x] 打印、续打
- [x] fix ueditor表格bug
- [x] 医学公式
- [x] 丰富的接口：鼠标标签、所有标签、修改、校验、合并打印等几十个接口，满足各种场景

## [开发环境]
> 基于UEditor1.4.3<br>
>UEditor是由百度web前端研发部开发所见即所得富文本web编辑器，具有轻量，可定制，注重用户体验等特点，开源基于MIT协议，允许自由使用和修改代码。
